Panduan Pemakaian

Kumpulan panduan langkah-demi-langkah buat pakai Sellvix AI, dari download data di Shopee sampai upload ke aplikasi ini.

1. Cara Download Laporan dari Shopee

📦 Data Penjualan (Order All)
  1. Login ke seller.shopee.co.id
  2. Menu Pesanan → Pesanan Saya
  3. Klik tab "Semua"
  4. Atur filter tanggal berdasarkan "Waktu Pesanan Dibuat" sesuai periode yang mau diambil
  5. Klik tombol "Export" (biasanya di kanan atas)
  6. Klik Export lagi di jendela konfirmasi, tunggu diproses
  7. Buka Riwayat Download, klik Download buat ambil file Excel-nya
📢 Data Iklan (Laporan CPC)
  1. Dari Seller Center, klik menu Iklan Shopee (atau ke iklan.shopee.co.id langsung)
  2. Masuk ke "Semua Statistik Iklan" / tab Laporan
  3. Atur periode tanggal (maksimal 90 hari terakhir)
  4. Klik tombol "Unduh Laporan"
  5. Pilih "Semua Laporan Iklan CPC", download sebagai CSV/Excel
💰 Laporan Penghasilan (Income Sudah Dilepas)
  1. Dari Seller Center, menu Keuangan → Penghasilan Saya
  2. Verifikasi password kalau diminta
  3. Buka "Catatan Transaksi Penghasilan", pilih periode (mingguan/bulanan)
  4. Klik ikon download di sebelah laporan yang mau diambil
🏦 Laporan Saldo (Balance Transaction Report) — opsional
  1. Dari Seller Center, menu Keuangan → Saldo Saya
  2. Buka tab "Laporan Transaksi"
  3. Atur rentang tanggal, klik Export/Download

Ini opsional — cuma dipakai buat kartu "Dana Ditarik ke Bank" di Laporan Keuangan.

2. Cara Upload File Excel ke Aplikasi

  1. Buka menu Upload Pusat (tab pertama di navigasi atas)
  2. Klik kotak "Klik untuk pilih file, atau seret & lepas di sini", atau langsung drag file dari folder Download HP/PC kamu ke kotak itu
  3. Bisa pilih banyak file sekaligus — Order All, Laporan Iklan, Laporan Penghasilan, Laporan Saldo, semua boleh diupload bareng
  4. Sistem otomatis mendeteksi jenis tiap file dari isinya, lalu memprosesnya masing-masing
  5. Progress & hasil tiap file muncul langsung, tersimpan juga di tabel Riwayat Upload di bawahnya

Nama file bebas — aplikasi baca dari isi data, bukan nama file. Upload ulang file yang sama juga aman, tidak bikin data dobel (sistem otomatis menimpa data lama).

3. Cara Setting Produk & HPP (wajib sebelum upload pertama kali)

  1. Buka menu Laporan Keuangan, klik tombol "⚙️ Setting Produk & HPP per Series"
  2. Isi tabel: Nama Produk/Series (misal "Aulia"), SKU/Kata Kunci (kode SKU asli dari Shopee, atau kata yang muncul di nama produk), dan HPP per pcs (modal produksi)
  3. Bisa tambah manual satu-satu, atau klik "📥 Download Template" buat dapat file Excel siap isi, lalu upload lewat "↑ Upload Excel" buat isi banyak sekaligus
  4. Klik "💾 Simpan Semua"

Ini wajib diisi sebelum upload Data Penjualan pertama kali — kalau belum ada daftar produknya, semua item bakal masuk kategori "Tidak Dikenali".

4. Cara Menyimpan Laporan Aplikasi (Download PDF)

Laporan Keuangan bisa disimpan jadi file PDF — bisa dikirim ke investor, mitra, atau akuntan tanpa perlu kasih akses login ke aplikasi ini. Ada 2 tombol terpisah, tergantung kebutuhan.

📄 Download PDF (langsung tersimpan)
  1. Buka menu Laporan Keuangan, pilih periode yang mau disimpan
  2. Klik tombol "📄 Download PDF" di atas halaman
  3. Tunggu beberapa detik — file PDF langsung tersimpan otomatis ke folder Download HP/PC, tanpa jendela tambahan
🖨️ Print (buat print fisik ke kertas)
  1. Klik tombol "🖨️ Print" di sebelah tombol Download PDF
  2. Muncul jendela Print bawaan browser — pilih printer fisik kamu di bagian "Tujuan/Destination", lalu klik Print
  3. Kalau mau simpan sebagai PDF juga bisa lewat sini, pilih "Save as PDF" di Destination — tapi buat itu, tombol "📄 Download PDF" di atas lebih cepat (1 klik langsung)

PDF yang dihasilkan otomatis cuma berisi ringkasan KPI & tabel laba per series — tombol, menu navigasi, dan pengaturan nggak ikut kebawa, jadi hasilnya rapi buat dibagikan.

5. Cara Daftar & Masuk ke Aplikasi

  1. Buka halaman login — kalau belum punya akun, klik "Daftar di sini"
  2. Isi Nama Toko, Email, dan Password, klik Daftar
  3. Kalau sudah punya akun, isi Email & Password lalu klik Masuk — atau klik "Masuk dengan Google" buat login lebih cepat pakai akun Google
  4. Tiap akun (toko) datanya terpisah otomatis dari akun lain — nggak akan tercampur

Lupa password? Untuk saat ini, hubungi Chat Admin (ikon 💬 di pojok kanan bawah) buat bantuan reset akun.
